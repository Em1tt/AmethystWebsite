Registration & Logging in

This article will teach you the following:

  • Why servers use plugins like AuthMe
  • How to register using AuthMe
  • How to login using AuthMe
  • How to restart your password using AuthMe

Why servers use plugins like AuthMe

Servers usually use this type of plugins to protect their players. If a server allows cracked clients, and you played as "Em1t", anyone could use the name "Em1t" and play with your progress. That's why servers use AuthMe. If you create a strong password, people can't steal your account, because they do not know your password

Please keep in mind that servers using AuthMe automatically encrypt your password with one of the following:

  • MD5
  • SHA1
  • SHA256
  • WHIRLPOOL
  • XAUTH

How to register using AuthMe

Upon joining a server for the first time, which is using AuthMe, you will need to register.
You can achieve that with /register <yourPassword> <confirmPassword>
Where <yourPassword> is the password you wish to use for logging into the server
and <confirmPassword> is the same as <yourPassword>

After completing this step, you should be able to play on most servers!
